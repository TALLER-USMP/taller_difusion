# Introducción

Este proyecto forma parte del curso "Taller de Proyectos 2025 II", correspondiente al 8vo ciclo.

El sistema que se desarrollará busca automatizar y facilitar la gestión académica en instituciones educativas. A través de este sistema se podrá gestionar eficientemente documentos como sílabos, planes de estudio, esquemas de evaluación, y más.

El desarrollo se realiza bajo el enfoque de Aprendizaje Basado en Proyectos (ABP) y siguiendo metodologías ágiles como Scrum. La solución será construida de forma incremental en base a roles definidos dentro del equipo y objetivos semanales.

Este repositorio contiene el desarrollo del sitio web de difusión del proyecto, liderado por el rol de "Líder de colaboración y difusión".
