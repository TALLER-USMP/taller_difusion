# Web de Difusión - Proyecto ABP 2025-II

Esta es la web oficial de **seguimiento y difusión** del proyecto desarrollado en el curso **Taller de Proyectos - 2025 II** (8.º ciclo, USMP - Ingeniería de Software). El propósito de este sitio es centralizar la documentación, avances semanales, diagramas arquitectónicos y recursos generados durante el desarrollo del sistema principal.

## 🧠 Proyecto Principal

**Nombre del proyecto:** Sistema de Gestión Académica  
**Repositorio principal:** [taller_2025_2](https://github.com/TALLER-USMP/taller_2025_2)

Este proyecto forma parte de una metodología de Aprendizaje Basado en Proyectos (ABP), donde distintos roles colaboran para desarrollar una solución completa, incluyendo diseño, desarrollo, documentación y despliegue.

## 🌐 Objetivo de esta Web

Esta web no es el sistema final, sino una **herramienta de difusión, seguimiento y documentación**. Aquí podrás:

- Visualizar los avances semanales del equipo.
- Leer la documentación técnica y organizacional.
- Explorar los diagramas generados con Mermaid.js.
- Consultar el historial del proyecto por semanas.
- Acceder a recursos clave como informes, capturas y más.

## 🛠️ Tecnologías Utilizadas

- [Vite](https://vitejs.dev/) + [React](https://react.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- [Mermaid.js](https://mermaid.js.org/) para visualización de diagramas
- [React Router](https://reactrouter.com/) para el enrutamiento
- JavaScript con SWC (usando la plantilla moderna)

## 🚀 Cómo ejecutar la web localmente

```bash
# Clona este repositorio (o el tuyo si es un fork)
git clone https://github.com/TU_USUARIO/taller_difusion.git
cd taller_difusion

# Instala las dependencias
npm install

# Inicia el servidor de desarrollo
npm run dev
